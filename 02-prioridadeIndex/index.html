Por que index?

index.html

Ao entrarmos na internet para buscarmos um site, os mecanismos de
busca vão buscar no site procurado a existência de um arquivo index.

Caso a extensão seja HTML, o buscardor vai apresentá-lo logo de
início. Caso seja outra extensão, no caso PHP, o buscador vai
interpretá-lo e logo em seguida apresentar seu resultado.

Caso ocorra de termos os 2 casos em simultâneo, são as configurações
do próprio servidor que irão determinar qual index terá prioridade e será renderizado.
O correto é nunca ter 2 index na mesma pasta.

Um caso também muito ruim é NÃO HAVER NENHUM index no
projeto, o que pode renderizar para o cliente toda a estrutura do
projeto até sua root(raíz).
